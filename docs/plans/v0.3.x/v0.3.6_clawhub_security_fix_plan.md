# v0.3.6 修正プラン: ClawHub Security Scan 改善（Benign 化）

**Date:** 2026-04-07
**Severity:** HIGH
**Target Version:** v0.3.6

## 1. 問題の概要

ClawHub Security Scan が **Suspicious (Medium Confidence)** を返している。主な原因は「宣言内容と実装の不一致（Coherence Gaps）」と「不透明な表現」。

| 項目 | スキャナーの指摘 | 原因 |
|------|----------------|------|
| **Purpose & Capability** | Registry が "instruction-only" だが、実際はコード + Go バイナリ + postinstall | ClawHub 上のメタタイプ設定ミス |
| **Instruction Scope** | "stealthy hooks", "secretly inject" などの表現 | README.md のマーケティング表現がセキュリティリスクと誤認識される |
| **Install Mechanism** | postinstall が GitHub からバイナリをダウンロード | 一般的な挙動だが、宣言されていないため不審に思われる |
| **Credentials** | `GEMINI_API_KEY` を使用するが、宣言がない | メタデータ/ドキュメントへの記載漏れ |

### 2.2. `SKILL.md` の新規作成

ClawHub スキャナーが最も重視するファイル。パッケージのルートに配置し、以下の情報を明示する。

```yaml
---
name: episodic-claw
description: Local episodic memory engine with Go sidecar for Gemini embeddings
homepage: https://github.com/YoshiaKefasu/episodic-claw
metadata: |
  {
    "clawdbot": {
      "requires": {
        "bins": ["go"],
        "env": ["GEMINI_API_KEY"]
      },
      "primaryEnv": "GEMINI_API_KEY",
      "type": "runtime"
    }
  }
---
# episodic-claw

Local episodic memory engine for OpenClaw.

## Architecture
- **TypeScript**: Plugin interface, OpenClaw hooks, tool definitions.
- **Go Sidecar**: High-performance embedding (Gemini API), vector search (HNSW), lexical search (Bleve), and Pebble DB storage.

## Installation
This package includes a `postinstall` script that downloads the prebuilt Go sidecar binary from GitHub Releases.
- To skip the download, set `EPISODIC_SKIP_POSTINSTALL=1`.
- The binary is required for embedding and search functionality.

## Credentials
- **`GEMINI_API_KEY`**: Required for generating embeddings via Google's Gemini API. The Go sidecar reads this from the environment.

## Behavior
- **Memory Injection**: Automatically retrieves relevant past episodes and prepends them to the system prompt before each AI response.
- **File Operations**: Reads/writes episode files under `~/.openclaw/workspace/episodes/`.
- **Anchor Management**: Creates and consumes `anchor.md` to preserve context across compaction cycles.
```

### 2.3. `openclaw.plugin.json` へのメタデータ追加

スキャナーが `package.json` や `openclaw.plugin.json` を解析する際に、環境変数の要求を認識できるよう `metadata` フィールドを追加する。

```json
"metadata": {
  "clawdbot": {
    "requires": {
      "env": ["GEMINI_API_KEY"]
    },
    "primaryEnv": "GEMINI_API_KEY",
    "type": "runtime"
  }
}
```

### 2.4. ClawHub Web UI での設定更新（ユーザー作業）

コード修正だけでは解決しない項目。ClawHub のプラグイン編集画面で以下を更新する必要がある：
1.  **Type**: `instruction-only` → `runtime` (または `code-plugin`)
2.  **Environment Variables**: `GEMINI_API_KEY` を宣言
3.  **Postinstall**: 「Downloads Go binary from GitHub Releases」と明記
